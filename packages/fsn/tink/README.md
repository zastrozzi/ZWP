# Zastrozzi Financial Services Network Tink Library

Brief description of the Zastrozzi Financial Services Network Tink Library library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/fsn.tink';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build fsn.tink` to build the library.

## Running unit tests

Run `nx test fsn.tink` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
