# Zastrozzi Financial Services Network Mastercard Library

Brief description of the Zastrozzi Financial Services Network Mastercard Library library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/fsn.mastercard';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build fsn.mastercard` to build the library.

## Running unit tests

Run `nx test fsn.mastercard` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
