# Zastrozzi UKGovernment HMRC Library

Brief description of the Zastrozzi UKGovernment HMRC Library library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/ukgov.hmrc';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build ukgov.hmrc` to build the library.

## Running unit tests

Run `nx test ukgov.hmrc` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
