# platform.files

Brief description of the platform.files library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.files';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.files` to build the library.

## Running unit tests

Run `nx test platform.files` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
