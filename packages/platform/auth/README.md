# platform.auth

Brief description of the platform.auth library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.auth';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.auth` to build the library.

## Running unit tests

Run `nx test platform.auth` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
