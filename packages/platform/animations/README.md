# animations

Brief description of the animations library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.animations';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.animations` to build the library.

## Running unit tests

Run `nx test platform.animations` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
