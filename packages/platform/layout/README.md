# layout

Brief description of the layout library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.layout';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.layout` to build the library.

## Running unit tests

Run `nx test platform.layout` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
