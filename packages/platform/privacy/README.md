# privacy

Brief description of the privacy library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.privacy';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.privacy` to build the library.

## Running unit tests

Run `nx test platform.privacy` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
