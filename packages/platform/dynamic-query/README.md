# platform.dynamic-query

Brief description of the platform.dynamic-query library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.dynamic-query';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.dynamic-query` to build the library.

## Running unit tests

Run `nx test platform.dynamic-query` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
