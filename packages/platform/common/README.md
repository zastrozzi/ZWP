# common

Brief description of the common library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.common';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.common` to build the library.

## Running unit tests

Run `nx test platform.common` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
