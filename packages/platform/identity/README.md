# platform.identity

Brief description of the platform.identity library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.identity';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.identity` to build the library.

## Running unit tests

Run `nx test platform.identity` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
