# platform.flows

Brief description of the platform.flows library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.flows';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.flows` to build the library.

## Running unit tests

Run `nx test platform.flows` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
