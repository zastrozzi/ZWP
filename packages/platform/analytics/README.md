# analytics

Brief description of the analytics library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.analytics';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.analytics` to build the library.

## Running unit tests

Run `nx test platform.analytics` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
