# platform.google-cloud

Brief description of the platform.google-cloud library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/platform.google-cloud';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build platform.google-cloud` to build the library.

## Running unit tests

Run `nx test platform.google-cloud` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
