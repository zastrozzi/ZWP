# Zastrozzi Rewards Network Tillo Library

Brief description of the Zastrozzi Rewards Network Tillo Library library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/rewards-network.tillo';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build rewards-network.tillo` to build the library.

## Running unit tests

Run `nx test rewards-network.tillo` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
