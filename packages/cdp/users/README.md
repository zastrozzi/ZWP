# Zastrozzi CDP User Library

Brief description of the Zastrozzi CDP User Library library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/cdp.users';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build cdp.users` to build the library.

## Running unit tests

Run `nx test cdp.users` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
