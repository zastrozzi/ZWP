# Zastrozzi CDP Common Library

Brief description of the Zastrozzi CDP Common Library library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/cdp.common';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build cdp.common` to build the library.

## Running unit tests

Run `nx test cdp.common` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
