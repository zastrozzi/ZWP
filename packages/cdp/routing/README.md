# Zastrozzi CDP Routing Library

Brief description of the Zastrozzi CDP Routing Library library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/cdp.routing';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build cdp.routing` to build the library.

## Running unit tests

Run `nx test cdp.routing` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
