# cdp.partner-net

Brief description of the cdp.partner-net library.

## Usage

Install the package and import the module:

```ts
import { ExampleModule } from '@zwp/cdp.partner-net';
```

Add `ExampleModule` to the `imports` array of your Angular module.

## Building

Run `nx build cdp.partner-net` to build the library.

## Running unit tests

Run `nx test cdp.partner-net` to execute the unit tests.

## Configuration

This library does not require additional environment variables.
